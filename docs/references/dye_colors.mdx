# Dye Colors

In Atlas, anything that requires a color (such as wool, armor, etc.) must be a valid dye color.

| Color Name   | Color      | Hex Color |
| ------------ | ---------- | --------- |
| `white`      | White      | FFFFFF    |
| `orange`     | Orange     | D87F33    |
| `magenta`    | Magenta    | B24CD8    |
| `light_blue` | Light Blue | 6699D8    |
| `yellow`     | Yellow     | E5E533    |
| `lime`       | Lime Green | 7FCC19    |
| `pink`       | Pink       | F27FA5    |
| `gray`       | Gray       | 4C4C4C    |
| `silver`     | Light Gray | 999999    |
| `cyan`       | Cyan       | 4C7F99    |
| `purple`     | Purple     | 7F3FB2    |
| `blue`       | Blue       | 334CB2    |
| `brown`      | Brown      | 664C33    |
| `green`      | Green      | 667F33    |
| `red`        | Red        | 993333    |
| `black`      | Black      | 191919    |

`Source Credit:` [Bukkit Dye Colors](https://hub.spigotmc.org/javadocs/bukkit/org/bukkit/DyeColor.html)
