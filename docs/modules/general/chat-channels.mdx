# Chat Channels `<channels>`

| Specification | Changes    |
| ------------- | ---------- |
| 1.0.2         | DEPRECATED |
| 1.0.3         | REMOVED    |

## Main Configuration

```xml

<channels team-chat="true" global-chat="true" />

```

##### Main Configuration Attributes

| Attribute     | Description                | Type                                                 | Default |
| ------------- | -------------------------- | ---------------------------------------------------- | ------- |
| `team-chat`   | If team chat is enabled.   | <span class="badge badge--primary">True/False</span> | `true`  |
| `global-chat` | If global chat is enabled. | <span class="badge badge--primary">True/False</span> | `true`  |
