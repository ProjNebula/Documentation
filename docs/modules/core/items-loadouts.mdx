# Items & Loadouts `<loadouts>`

You can use this module to create sets of items and other player attributes that can be given to players at different times.

## Root Loadout Element `<loadout>`

The root loadout element is used to define basic information about the loadout.

```xml

<loadouts>
  <loadout id="default">...</loadout>
</loadouts>

```

##### Root Loadout Element Attributes

| Attribute | Description | Type | Default |
| --- | --- | --- | --- |
| `id` | The globally unique ID of this feature. This can be used to reference this feature from other parts of the XML. | <span class="badge badge--primary">Feature ID</span> | |
| `parent` | The id of the parent loadout for reference. | <span class="badge badge--primary">Parent Loadout ID</span> | |
| `force` | If the items should be forced into already occupied item slots. | <span class="badge badge--primary">True/False</span> | `false` |

### Compass `<compass>`

Set the player's compass target to a fixed position. (The compass module will override this)

```xml

<loadout>
  ...
  <compass>1, 1, 1</compass>
</loadout>

```

#### Element Text

| Description | Type |
| --- | --- |
| The target of the compass. | <span class="badge badge--primary">X,Y,Z Vector</span> |

### Effects `<effects>`

Use this to apply potion effects directly to the player.

```xml

<loadout>
  <effect amplifier="3" duration="5s">invisibility</effect>
  <effect amplifier="3" duration="5s">increase_damage</effect>
  <effect amplifier="3" duration="5s">regeneration</effect>
  <effect amplifier="3" duration="5s">speed</effect>
</loadout>

```

#### Element Text

| Description | Type |
| --- | --- |
| The type of the effect that the potion will cause. | <span class="badge badge--primary">PotionEffectType</span><br />[Possible Values](https://hub.spigotmc.org/javadocs/spigot/org/bukkit/potion/PotionEffectType.html) |

##### Effects Attributes

| Attribute | Description | Type | Default |
| --- | --- | --- | --- |
| `amplifier` | The amplifier of the potion effect. | <span class="badge badge--primary">Number (Without Decimal)</span> | `1` |
| `duration` | The duration of the effect. This attribute denotes a period of time and can either be in seconds or in a period format. Formats are as follows:     d - days     h - hours     m - minutes     s - seconds These can be combined to create any period. No spaces are allowed. | <span class="badge badge--primary">Duration</span> | `1` |

### Exhaustion `<exhaustion>`

Set the player's level of exhaustion (This ranges from 0.0 to 4.0. When the exhaustion level reaches above 4.0 it will get subtracted by 4.0 and subtracts 1 point of hunger or saturation.)

```xml

<loadout>
  ...
  <exhaustion action="multiply">2</exhaustion>
</loadout>

```

#### Element Text

| Description | Type |
| --- | --- |
| The exhaustion modifier that should be used by the action. | <span class="badge badge--primary">Number (Without Decimal)</span> |

##### Exhaustion Attributes

| Attribute | Description | Type | Default |
| --- | --- | --- | --- |
| `action` | The action which is applied to the player's exhaustion level. | <span class="badge badge--primary">Number Action</span><br />`none - Perform no action and keep the player's exhaustion level the same.`, `set - Set the player's exhaustion level to the supplied number.`, `add - Add the supplied number to the player's exhaustion level.`, `subtract - Subtract the supplied number from the player's exhaustion level.`, `multiply - Multiply the supplied number with the player's exhaustion level.`, `divide - Divide the player's exhaustion level by the supplied number.`, `power - Raise the player's exhaustion level by the supplied number.` | `set` |

### Generic Items `<boots>``<chestplate>``<helmet>``<item>``<kit-menu>``<leggings>``<shop-opener>`

Items can have many attributes, and are the main element of loadouts.

```xml

<loadout id="default">
  <item slot="0" material="stone sword" />
  <item slot="1" material="bow" />
</loadout>

```

##### Generic Items Attributes

| Attribute | Description | Type | Default |
| --- | --- | --- | --- |
| `slot` | Slot in the inventory where the item should be placed. (If no slot is specified, the item will be put in the first available slot) | <span class="badge badge--primary">Number (Without Decimal)</span> | |
| `material` | The material of the item. | <span class="badge badge--primary">Material</span><br />[Possible Values](https://hub.spigotmc.org/javadocs/spigot/org/bukkit/Material.html) | |
| `shop-id` | If a shop opener, which shop this item should open on right click. The id of the shop for reference. | <span class="badge badge--primary">Shop ID</span> | |
| `damage` | The damage value of the item. (This can be used to color wool or to change the durability of a pickaxe.) | <span class="badge badge--primary">Number (Without Decimal)</span> | |
| `amount` | The amount of the item to be given to the player. (Use -1 for infinite stacks) | <span class="badge badge--primary">Number (Without Decimal)</span> | `1` |
| `unbreakable` | If the item should never run out of durability. | <span class="badge badge--primary">True/False</span> | `false` |
| `flags` | The flags that should be applied to the item. Separate each flag with a , | <span class="badge badge--primary">Text</span> | |
| `color` | The color to be applied to the item. (This item must be a colorable item, or a parser error will be thrown) Colors can be defined in 2 different ways. If the color starts with a '#', the hex of the color will be parsed (use this for exact colors) If not, a generic color (red, green, blue, etc) should be used. | <span class="badge badge--primary">Color</span> | |
| `team-color` | If the player's team color should be used to color the item. This will override the other color attribute. | <span class="badge badge--primary">True/False</span> | `false` |
| `potion` | The type of the potion that will be displayed. (This will not override the effects added by the effects tag) | <span class="badge badge--primary">PotionType</span><br />[Possible Values](https://hub.spigotmc.org/javadocs/spigot/org/bukkit/potion/PotionType.html) | |
| `projectile` | The id of the projectile for reference. | <span class="badge badge--primary">Projectile ID</span> | |
| `grenade` | If the item should behave like a grenade projectile, creating an explosion when landing. | <span class="badge badge--primary">True/False</span> | `false` |
| `grenade-power` | The power of the grenade projectile (4.0 is equivalent to TNT). | <span class="badge badge--primary">Number</span> | `1.0` |
| `grenade-fire` | Whether the grenade projectile should create fire on landing. | <span class="badge badge--primary">True/False</span> | `false` |
| `grenade-destroy` | Whether the grenade projectile should damage blocks on landing. | <span class="badge badge--primary">True/False</span> | `false` |
