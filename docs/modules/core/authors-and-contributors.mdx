# Authors and Contributors `<authors>`

The authors element is provided to give credit to the people that made contributions to the creation of a map. Their minecraft usernames are listed in a variety of locations while the map is being played.

## Author `<author>`

An author is a person who contributed to the map in some way.

```xml

<authors>
  <author uuid="069a79f444e94726a5befca90e38aaf5" />
  <!-- Notch -->
  <author uuid="8242e55ecc6a458ab272a2b34918045f" role="Assistance with map design." />
  <!-- funkystudios -->
</authors>

```

##### Author Attributes

| Attribute     | Description                                                         | Type                                                     | Default |
| ------------- | ------------------------------------------------------------------- | -------------------------------------------------------- | ------- |
| `contributor` | If a player should be counted as a main author or as a contributor. | <span class="badge badge--primary">True/False</span>     | `false` |
| `uuid`        | The UUID of the player that should be credited.                     | <span class="badge badge--primary">Minecraft UUID</span> |         |
| `role`        | The specific task that the user performed.                          | <span class="badge badge--primary">Text</span>           |         |
| `promo`       | A link to the player's website.                                     | <span class="badge badge--primary">URL</span>            |         |

## Organisation `<organization>`

An organisation represents a group of people who helped with the map.

```xml

<authors>
  <organization name="The Avicus Network" role="Server" promo="https://avicus.net" />
</authors>

```

##### Organisation Attributes

| Attribute | Description                                 | Type                                           |
| --------- | ------------------------------------------- | ---------------------------------------------- |
| `name`    | The name of the organisation.               | <span class="badge badge--primary">Text</span> |
| `role`    | The specific task that the group performed. | <span class="badge badge--primary">Text</span> |
| `promo`   | A link to the group's website.              | <span class="badge badge--primary">URL</span>  |
